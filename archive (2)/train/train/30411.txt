ವಲ್ಲಭಾಚಾರ್ಯರ ಮತದಲ್ಲಿ ಜೀವನು ಪರಮಾತ್ಮನ ಪರಿಣಾಮ . ಪರಮಾತ್ಮನು ವಿಹರಿಸಲು ಬಯಕೆಯಾದಾಗ ಜೀವರೂಪ ತಳೆಯುತ್ತಾನೆ. ಆಗ ಪರಮಾತ್ಮನ ವಿಶೇಷಗಣಗಳು ಮರೆಯಾಗುತ್ತವೆ. ದೀನತೆ ಅಜ್ಞಾನ ಮುಂತಾದವು ಉಂಟಾಗುತ್ತವೆ. ಜೀವನು/ರು ಜ್ಞಾತೃ, ಜ್ಞಾನ ರೂಪಿ ; ಅಣುರೂಪಿ ; ಮುಕ್ತವಾದಾಗ ಅವು ಬ್ರಹ್ಮನಲ್ಲಿ ಸೇರಿಹೋಗುತ್ತವೆ - ಅಭಿನ್ನವಾಗುತ್ತವೆ. ಭಕ್ತಿಯೀಂದ ಮೋಕ್ಷ ದೊರಕುವುದು . 
ಈ ದರ್ಶನದಲ್ಲಿ ಆತ್ಮನು ಶುದ್ಧ ಚೈತನ್ಯ. ಅವನು ಅಪರಿಮಿತ ಜ್ಞಾನ ಶಕ್ತಿಯುಳ್ಳವನು . ಶಿವ ಸ್ವರೂಪಿ ; ಅವ್ಯಕ್ತ ; ಮಲಗಳ ಸಂಯೋಗ ಕಾರಣದಿಂದ ಅವನ ಶಕ್ತಿ ಸಂಕೋಚ ವಾಗಿದೆ . ಜೀವನು ಸಹಜ ಸ್ವರೂಪ ಪಡೆದನೆಂದರೆ ಮುಕ್ತಿ . 

ಚಾರ್ವಾಕ ದರ್ಶನ ; ಜೈನ ಧರ್ಮ - ಜೈನ ದರ್ಶನ ; ಬೌದ್ಧ ಧರ್ಮ ; ಸಾಂಖ್ಯ-ಸಾಂಖ್ಯ ದರ್ಶನ ; ->ರಾಜಯೋಗ ; ನ್ಯಾಯ ದರ್ಶನ ; ವೈಶೇಷಿಕ ದರ್ಶನ;; ಮೀಮಾಂಸ ದರ್ಶನ - ; ವೇದಾಂತ ದರ್ಶನ / ಉತ್ತರ ಮೀಮಾಂಸಾ ; ಅದ್ವೈತ ; ಆದಿ ಶಂಕರರು ಮತ್ತು ಅದ್ವೈತ ; ವಿಶಿಷ್ಟಾದ್ವೈತ ದರ್ಶನ ; ದ್ವೈತ ದರ್ಶನ - ಮಾಧ್ವ ಸಿದ್ಧಾಂತ ; ಪಂಚ ಕೋಶ ; ಶ್ರೀಮನ್ಮಹಾಭಾರತಮ್ ಮತ್ತು ದ್ವೈತ ದರ್ಶನ ; ವೀರಶೈವ; ಬಸವಣ್ಣ; ಭಾರತೀಯ ತತ್ವಶಾಸ್ತ್ರ ಸಮೀಕ್ಷೆ; ಭಗವದ್ಗೀತಾ ತಾತ್ಪರ್ಯ ; ಕರ್ಮ ಸಿದ್ಧಾಂತ ; ವೇದ--ಭಾರತೀಯ ತತ್ತ್ವಶಾಸ್ತ್ರದಲ್ಲಿ ದೇವರುಭಾರತೀಯ ತತ್ತ್ವಶಾಸ್ತ್ರದಲ್ಲಿ ಜಗತ್ತು-ಅಸ್ತಿತ್ವ-ಸತ್ಯವೇ-ಮಿಥ್ಯವೇ -;ಭಾರತೀಯ ತತ್ತ್ವಶಾಸ್ತ್ರದಲ್ಲಿ ಜೀವಾತ್ಮ --ಮೋಕ್ಷ- ಭಾರತೀಯ ತತ್ತ್ವಶಾಸ್ತ್ರದಲ್ಲಿ ಮೋಕ್ಷ
